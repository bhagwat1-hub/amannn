# AMAN Superbrain AI

A modular, voice-first Python assistant that combines GPT reasoning, Whisper speech recognition, text-to-speech, SQLite memory, desktop automation, and YOLO-based vision.

## Run

```bash
cd aman_superbrain
pip install -r requirements.txt
python main.py
```

The startup voice message is:

> "AMAN Superbrain activated. How can I assist you?"
